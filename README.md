# Library-management-System
📚 Library Management System
A beginner-friendly Python Library Management System that allows users to manage books, register users, lend books, and return borrowed books using a simple console-based interface.
✨ Features
➕ Add new books
📖 View available books
❌ Remove books
👤 Add library users
📋 View registered users
📚 Lend books to users
🔄 Return borrowed books
📌 Track borrowed books
🖥️ Simple menu-driven interface
🛠️ Technologies Used
Python
Dictionary
Lists
Functions
Loops
Conditional Statements
🚀 How It Works
The system maintains:
A list of available books
A list of registered users
A dictionary to track borrowed books
Users can:
Add books to the library
Register library users
Lend books to users
Return borrowed books
View all available and borrowed books
▶️ Run the Program
Install Python on your system
Save the code as library_management.py
Open terminal or command prompt
Run the program using:
Bash
python library_management.py
